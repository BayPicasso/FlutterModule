Store Request Tutorial

Store Request adalah permintaan pengambilan barang dari gudang (store) oleh departemen lain.

Buka Module Request Online

• Klik Button Transaction

• Pilih (SR) Store Request


Maka nanti Menu Utamanya seperti ini

• Untuk Membuat Store Request kita bisa Klik Button New


Maka nanti Form Utamanya seperti ini

• Kita isi Description dari SR tersebut

• Kemudian di Department To kita isi Departmentnya

• Warehouse From kita isi Warehouse yang ingin dituju

• Notes bisa diisi jika ada Catatan tambahan

• Kalau sudah kita bisa klik Save


Lalu akan muncul Form untuk mengisi Productnya

• Kita ketik saja dibagian Product Name untuk memilih Productnya

• Dibagian Unit kita sesuaikan

• Lalu di Qty kita isi kuantiti barang yang diinginkan

• Department dan Warehouse sudah otomatis terisi sama dengan Form yang Pertama

• Di COA/Account tidak perlu diubah, sudah default mengikuti mappingan

• Notes kita bisa isi jika ada Catatan tambahan

• Jika ada tambahan Product/Barang kita bisa Klik Apply, jika sudah sesuai kita bisa Klik OK dan Close


Jika sudah selesai, Kita bisa Send SR kita agar nanti bisa di Approve

• Kita pastikan untuk Klik tombol Send agar SR kita bisa ditindak lanjuti
