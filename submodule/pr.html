Purchasing Request Tutorial

Purchasing Request adalah permintaan pembelian barang atau jasa yang dibuat oleh departemen tertentu kepada bagian pembelian (purchasing).

Buka Module Request Online

• Klik Button Transaction

• Pilih (PR) Purchasing Request


Maka nanti Menu Utamanya seperti ini

• Untuk Membuat Purchasing Request kita bisa Klik Button New


Maka nanti Form Utamanya seperti ini

• Kita isi Deskripsinya

• Department To kita isi Departmentnya

• Warehouse To kita isi Warehousenya dengan catatan jika ingin dijadikan Stock (Inventory), kalau dikosongi maka akan menjadi Expenses (langsung pakai)

• Delivery Date kita bisa isi tanggal berapa barang itu diharapkan datang

• Notes diisi jika ada Catatan tambahan terkait PR tersebut

• Kalau sudah kita Klik Save


Setelah itu Kita isi Item/Productnya

• Dibagian Product Name kita ketik saja Barang yang ingin di Request maka nanti akan muncul Lookup dari nama barangnya

• Kemudian di Unit kita isi Unit barangnya

• Qty kita isi kuantiti barang yang diminta

• Department To sudah otomatis terisi sama dengan Department yang di From awal

• Account sudah otomatis ter default

• Notes kita bisa isi Catatan tambahan jika ada

• Jika ingin menambahkan Product lainnya kita bisa klik Apply untuk menyimpan sekaligus mengisi Form Productnya kembali untuk menambahkan Barang lagi

• Kalau sudah kita bisa Klik Button OK, dan kita bisa Close


Jika sudah selesai, Kita bisa Send PR kita agar nanti bisa di Approve

• Kita pastikan untuk Klik tombol Send agar PR kita bisa ditindak lanjuti
