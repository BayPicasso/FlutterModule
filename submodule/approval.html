Approval Tutorial

Approval adalah program untuk melakukan Approve kepada Permintaan pembelian yang dibuat oleh User Department, biasanya Approve dilakukan oleh Purchasing, Department Head, hingga DOF dan GM.

Buka Module Approval

Maka nanti Menu Utama dari Approval seperti ini

- Untuk melakukan Approve pada Transaksi pertama tama kita bisa masuk ke Outstanding

- Di Outstanding ini kita bisa melihat transaksi apa saja yang belum di Approve oleh kita

- Untuk melakukan Approve sebenarnya ada 2 cara

Cara Pertama

Kita bisa Checklist dulu untuk transaksi mana yang ini di Approve, untuk checklist kita bisa beberapa transaksi sekaligus -

Lalu akan muncul Menu dibawah yaitu : -

Approve untuk menyetujui -

Reject untuk menolak -

Return untuk mengembalikan -

Return disini maksudnya transaksi yang di return akan kembali ke Level Approval sebelumnya atau akan kembali ke User yang membuat -

Cara Kedua

- Untuk cara kedua ini kita bisa pilih dulu transaksinya lalu klik Tombol See Details

- Maka akan muncul popup untuk detail Item/Product dari transaksi itu

- Untuk Approve, Return, Reject kita bisa perhatikan tombol di kanan bawah

Di See Details kita juga bisa melakukan penyesuaian

Kita bisa mengedit Qty (kuantiti) dari item itu dengan klik tombol Pensil -

Atau ingin mengcancel item itu dengan klik tombol Tong Sampah berwarna merah -

Button

- Ok, untuk penjelasan tombol tombol di bagian atas ini adalah

- Transaction Type kita bisa filter transaksi jenis apa yang ingin ditampilkan

- Replace for ini adalah untuk Approval pengganti, jadi semisal user Dept Head belum bisa mengapprove maka user SPV atau user lainnya bisa menggantikannya (tergantung settingan)

- Print From adalah tombol untuk Print Form dari transaksi yang dipilih

- Filter adalah tombol untuk memfilter transaksi apa yang ingin ditampilkan semisal ingin memunculkan transaksi dari Department tertentu
